# 测试大模型编程能力的有趣项目

## 入门级（单文件、逻辑驱动）

1. **生命游戏 (Conway's Game of Life)** — 测试网格逻辑、状态管理
2. **2048 游戏** — 测试矩阵操作、动画、交互逻辑
3. **贪吃蛇 / 俄罗斯方块** — 经典小游戏，考验实时渲染和碰撞检测
4. **Markdown 解析器** — 测试字符串处理、递归、AST 构建

## 中级（多模块、算法为主）

5. **解释器/计算器** — 实现一个小型编程语言或表达式求值器，测试解析和递归能力
6. **路径寻找可视化** — A*/Dijkstra 算法可视化，测算法实现和动画渲染
7. **简易数据库** — 支持 SQL 子集的内存数据库（SELECT/JOIN/WHERE）
8. **正则表达式引擎** — 从零实现正则匹配，NFA/DFA 构造

## 高级（系统设计、多文件协作）

9. **简易操作系统内核** — 在 RISC-V/x86 上实现最基础的 OS
10. **Redis 简易版** — 实现基本数据结构 + 网络协议 + 持久化
11. **自建 Web 框架** — 从 TCP socket 开始，实现 HTTP 解析 + 路由 + 模板引擎
12. **简易 Docker** — 用 Linux namespace/cgroup 实现容器隔离

## 创意型（趣味性强）

13. **物理模拟** — 弹簧、流体、布料模拟，考验数学和渲染
14. **音频可视化器** — Web Audio API + Canvas，测试多媒体处理
15. **光线追踪器 (Ray Tracer)** — 经典图形学项目，几百行代码出震撼效果
16. **代码高尔夫挑战** — 用最少的字符实现特定功能

## 推荐的测试方式

| 维度 | 适合测试的项目 |
|------|---------------|
| 算法能力 | A* 可视化、正则引擎、解释器 |
| 系统设计 | 简易数据库、Redis、Web 框架 |
| 前端/交互 | 2048、物理模拟、光线追踪 |
| 调试能力 | 给一个有 bug 的复杂项目让它修 |
| 创造力 | "用你喜欢的语言做一个让我惊讶的东西" |

## 社区流行的测试基准

- **SWE-bench**：让模型修真实 GitHub issue
- **HumanEval**：函数级编程题
- **给一个模糊需求**，看模型能不能主动澄清、设计方案、实现并测试
